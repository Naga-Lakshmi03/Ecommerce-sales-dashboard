# E-commerce Sales Analysis (Interactive Dashboard Creation Using Power BI)
## Project Overview 
Developed an interactive US E-Commerce Sales Dashboard in Power BI to analyse Year-to-Date sales, profit, product performance, regional trends, and shipping preferences. The dashboard provides actionable business insights through KPI tracking, YoY comparisons, and dynamic visualizations to support data-driven decision-making.

## Dataset Used
- <a href="https://github.com/Naga-Lakshmi03/Ecommerce-sales-dashboard/blob/main/ecommerce_data.csv">Data Set 1</a>
- <a href="https://github.com/Naga-Lakshmi03/Ecommerce-sales-dashboard/blob/main/us_state_long_lat_codes.csv">Data Set 2</a>
## Problem Statement 
An e-commerce company wants to understand its sales performance, customer purchasing behaviour, and profitability across different regions, categories, and time periods. Management needs insights to identify high-performing areas, improve weak-performing segments, and make data-driven business decisions.

## Key Performance Indicator’s
- Create a KPI Banner showing YTD Sales, YTD Profit, YTD Quantity sold, YTD Profit Margin.

- Find Year on Year growth for each KPI and show a YTD sparkline for each measure in the KPI to understand the monthly trend for each fact.

- Find YTD Sales, PYTD Sales, YOY Sales growth for different customer category. Add a trend icon for each category.

- Find YTD Sales Performance by each state.

- Top 5 and Bottom 5 Products by sales.

- YTD Sales by Region to know best and worst performing region all over country.

- YTD Sales by Shipping Type to get the best Shipping type percentage.
