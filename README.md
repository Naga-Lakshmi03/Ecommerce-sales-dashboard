# E-commerce Sales Analysis (Interactive Dashboard Creation Using Power BI)
## Project Overview 
Developed an interactive US E-Commerce Sales Dashboard in Power BI to analyse Year-to-Date sales, profit, product performance, regional trends, and shipping preferences. The dashboard provides actionable business insights through KPI tracking, YoY comparisons, and dynamic visualizations to support data-driven decision-making.
